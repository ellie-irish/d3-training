In a new HTML file, read in data.json file using D3. This file contains an array of 10 objects that have the following properties: size and color.

In the bottom half of the screen, create 10 circles (one for each object in the data) with fill colors equal to the `color` property on the corresponding data object. Now picture the screen as an Cartesian plane with (0, 0) at the upper-left corner of the screen. Position each circle such that the coordinates of its center are (`size`, -2*`size`).
